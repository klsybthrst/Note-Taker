# Note-Taker
This note taking application allows you to type and save notes with a title and text area. Using ExpressJs, the user can save and delete notes as they please.

# Technologies
<ul>
  <li>JavaScript</li>
  <li>NodeJS</li>
  <li>Express</li>
</ul>
